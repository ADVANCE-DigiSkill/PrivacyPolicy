In order to monitor the performance in our app, both in a technical sense and in a sense of usage, we may collect and send the following personal data to third parties if you consent to it in the opt-in dialog. The personal data collected is as follows:


Firebase Crashlytics (Examples of stored device information collected by Crashlytics)

- Crashlytics Installation UUIDs (An RFC-4122 UUID)

- Your device's operating system name and version number

- Your device's model name, CPU architecture, amount of RAM and disk space

- A boolean indicating whether the device was jailbroken/rooted


Google Analytics for Firebase (Google Analytics for Firebase data collection)

- Mobile ad IDs

- Android IDs

- Firebase installation IDs

- Analytics App Instance IDs


See Firebase Data Processing and Security Terms for more information.

Having accurate information about you permits us to provide you with a smooth, efficient, and customized experience. Specifically, we may use information collected via the Application to:

  1. Compile anonymous statistical data and analysis for use internally or with third parties.

  2. Increase the efficiency and operation of the Application.

  3. Monitor and analyze usage and trends to improve your experience with the Application.


Firebase Crashlytics

The Crashlytics service provided as a part of Google’s Firebase platform is used to catch errors in our App. If usage of the App at any time leads to unexpected errors or the App crashing, specific information, such as the device type, operating system version, date and time of the error, country in which the query originated, and language selected for the operating system, will be sent to Crashlytics if you have opted in. You can find more information about data protection at Crashlytics here: Crashlytics and App Distribution Data Processing and Security Terms


Google Analytics for Firebase

The Google Analytics service provided as part of Google’s Firebase platform is used to keep track of the user’s progression through the app. This enables us to understand things such as how the App is used, which parts of it require improvement, how many users finish the courses, and whether it is a success overall. You can find more information about data protection at Google Analytics for Firebase here: Firebase Data Processing and Security Terms
